# daviesDevelops

![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg)

## Description

daviesDevelops is my personal portfolio. A SPA built with React and includes a short bio, a contact page, a link to download my resume and portfolio.
I was motivated to create this application to showcase my portfolio online as well as easy access to my relevant information.

This application allows potential employers to easily view my portfolio and review my previous work to help assess if I would be the candidate to join there team.

## Technologies Used

HTML,CSS,JavaScript,React
